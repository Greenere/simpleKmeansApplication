# simpleKmeansApplication
A simple K-means application toy with graphic user interface

FinalKmeans.py is the main program, inside which lie both k-means algorithm part and GUI part of codes.
The environment of programming is Python 3.6-64 bit on windows 10.

fkmeansData.txt is the exemplary data used to demonstrate the kmeans clustering process.
It is an public data gathered online about the acreage, population and GDP of 49 cities in central China.

README.pdf is a vivid readme file explaining the usage of it.
